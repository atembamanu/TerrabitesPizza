# Terribites Pizza

![Alt text](assets/pizzaa.png?raw=true "")

## Description
This is a pizza place  customers order different sizes of pizzas with one or more toppings.

## Versioning

 TerrabitesPizza-V1.0 

## Author

* [**atembamanu**](https://github.com/atembamanu)

## Features


As a user of the web application you will be able to:

1. Allow customers to choose pizza sizes
2. Allow Customers to choose thr crust and toppings
3. Select your gender from the radio button choices.
4. Add pizza to the basket
5. See the basket
6. Order more pizzas
7. select deleivery method 
8. Checkout.

## Behaviour Driven Development (BDD)
|Behaviour 	           |    Input 	                 |       Output          |
|----------------------|:---------------------------:|----------------------:|       
|Page loads and pizzas are displayed  |User selects a pizza size    |The price of the corresponding pizza size is displayed|
|Page loads and crust select box is displayed  |User selects the crust they want   |The price of the corresponding pizza crust is displayed|
|Page loads and toppings select box is displayed  |User selects the toppings they want   |The price of the corresponding pizza topping is displayed|
